## 检测策略

### 逐行扫描法
从图片底部向上扫描，检测满足以下条件的行：
- 灰色像素占比 > 3%（gray_range: 40-190）
- 深色像素占比 < 15%（dark < 40）
- 连续 5 行以上满足条件 → 标记为水印行

### 色相检测法
- 正文文字：暗色（RGB 均值 < 40）
- 水印文字：中灰色（RGB 均值 40-180）
- 背景：亮色（RGB 均值 > 200）

### 位置启发式
- 水印通常在底部 10% 区域
- 页眉水印在顶部 5% 区域
- 边缘水印需要检测边缘 50px 条带

## 添加新水印模式

编辑 `config.json` 的 `postprocessing.watermark_keywords`：

```json
{
  "postprocessing": {
    "watermark_keywords": [
      "智慧的大聪明",
      "售后更新微信",
      "yize2288",
      "新发现的_水印关键词"
    ]
  }
}
```

或者调整检测参数：

```json
{
  "watermark": {
    "gray_range": [40, 190],
    "gray_ratio_threshold": 0.03,
    "dark_ratio_max": 0.15,
    "crop_bottom": 60
  }
}
```

## 自我学习

`self_improve.py` 会分析使用日志，自动优化 `crop_bottom` 等参数。
当新水印模式出现时，OCR 质量分数会下降，系统会建议调整阈值。
